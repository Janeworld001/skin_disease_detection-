# Skin Disease Detection App

A web-based application that detects skin diseases using a trained YOLO deep learning model. Users can upload an image of a skin condition, and the model predicts the disease class.

---

## Live Demo

https://skin-disease-detection-model.streamlit.app/

---

##  Features

* Upload skin images for analysis
* Real-time disease prediction
* YOLO-based deep learning model
* Simple and user-friendly interface

---

## Model

* Model: YOLO (Ultralytics)
* Trained on skin disease dataset
* File: `Correct_skin_disease_model.pt`

---

##  Tech Stack

* Python
* Streamlit
* Ultralytics (YOLO)
* OpenCV (headless)
* PyTorch

## ⚙️ Installation (Local Setup)

1. Clone the repository:

```
git clone https://github.com/Janeworld001/skin_disease_detection-.git
cd skin_disease_detection-
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
